# 📝 TaskFlow - Advanced React To-Do Application

Welcome to **TaskFlow**, a fully responsive and feature-rich task manager built with **React**, **Tailwind CSS**, **Redux**, and **OpenWeather API**. This project was built as part of the frontend training assessment to demonstrate skills in modern web development.

## 📦 Tech Stack

- React (with Hooks)
- Tailwind CSS** (Utility-first styling)
- Redux (Toolkit + Thunk)** for state management
- API for real-time weather info
- LocalStorage for persistent task and auth data
- Vite / CRA (choose based on your setup)

---

## ✨ Features

- 🔐 User Authentication** (Simulated login using Redux)
- 🧾 Add, Delete, and View Tasks**
- 🔺 Task Prioritization** (High / Medium / Low)
- 🌦️ Weather Integration** (Shows weather for task location)
- 📊 Real-Time Stats** (Total, Completed, Pending, High Priority)
- 💾 LocalStorage Persistence** (Tasks + Auth)
- 📱 Fully Responsive** (Mobile-first with Flex/Grid)
- 🧼 Minimal UI using Tailwind + MUI Buttons**

## 🔧 Installation & Setup

```bash
# Clone the repository
git clone https://github.com/your-username/taskflow-app.git
cd taskflow-app

# Install dependencies
npm install

# Create .env file
touch .env
```

### Add your OpenWeather API key in `.env`:

```env
VITE_WEATHER_API_KEY=your_api_key_here
```

Then run the app:

```bash
npm start
```
